1 - The type map

A dictionary, so called a map, allows to map 2 value between them. It is very similar to Arrays, except they aren't indexed with integer starting at 0, but with key of a given type.

package main

func main() {
	var dic [string]string = ["Nu": "Programming language"]
}

Here, the map dic associate 2 string together. To access element it is really similar as arrays

package main

func main() {
	quote := ["Hello There": "General Kenobi"]
	kenobi := "Hello There"
	grivious := quote[kenobi]
}

The operator [] return a pointer to the element associated to the key. If a key doesn't exist it is the nullptr (we'll see that later). To append an element to the map, we use the operator [], like when we access an element, except we put the key and the value, separated by ':'. It also gives the pair added.

package main

import Std io.console as cio

func main() {
	contacts := ["Elodie": "+33 6 23 13 54 37"]
	newOne := contacts["Matthew": "333 666 345"]
	cio.Println(newOne) // print struct{key: "Matthew", value: "333 666 345"}
}

Note that using it on an existing pair it will create a new reference instead of changing the existing one

package main

import Std io.console as cio

func main() {
	ages := ["Andrew": 18, "Carla": 23, "peter": 15, "julia": 21]
	andrewAge := &ages["Andrew"] // andrewAge = 18
	ages["Andrew"]++ // now andrewAge = 19
	ages["Andrew": 20] // andrewAge still = to 19 !
}

We'll retalk about that when talking about pointer.

2 - Operation on the map

.keys/.values to get an array of the keys/values

package main

import Std io.console as cio

func main() {
	contacts := ["Elodie": "+33 6 23 13 54 37"]
	contacts["Matthew": "333 666 345"]
	cio.Println(contacts.keys) // print [Elodie, Matthew]

.first/.last gives the first/last pair added

package main

import Std io.console as cio

func main() {
	contacts := ["Elodie": "+33 6 23 13 54 37"]
	contacts["Matthew": "333 666 345"]
	cio.Println(contacts.first) // struct{key: "Elodie", value: "+33 6 23 13 54 37"}

.delete(key K) to delete the pair associated to the given key, and return the associated pair

package main

import Std io.console as cio

func main() {
	contacts := ["Elodie": "+33 6 23 13 54 37"]
	contacts["Matthew": "333 666 345"]
	contacts.delete("Elodie")
}

.clear() delete all pair

.length gives the number of element

.has(key T) bool gives true if the given key is present, else false
